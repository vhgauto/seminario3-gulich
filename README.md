# Better maps with `R`

Repositorio de la presentación del Seminario #3 "Better Maps with R" del MS Víctor Gauto, para el **Insituto Gulich**.

# [Link a la presentación](https://vhgauto.github.io/seminario3-gulich/)

La creación de mapas es fundamental para la comunicación de datos GIS. La herramienta que usemos para confeccionarlos va a definir nuestras posibilidades y limitaciones. Usar lenguaje de programación permite asegurar la reproducibilidad, actualizar los resultados ante cambios y realizar grandes modificaciones en pocos pasos. Este seminario tiene como objetivo explorar diferentes formatos de mapas (estático, animado, interactivo, 3D), con ejemplos en lenguaje de programación `R`.
